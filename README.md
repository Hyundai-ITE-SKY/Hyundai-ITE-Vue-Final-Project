# MSA 기반 PWA 온라인 쇼핑몰 프로젝트 - Front

### 🗨️ **개요**

---

- 현대 IT&E 개발자 양성과정의 최종 프로젝트이다.
- 더 한섬(THE HANDSOME)에서 제공받은 상품 데이터를 활용하여 쇼핑몰 홈페이지를 제작한다.
- 상품 전시가 가능한 모바일 환경의 프론트를 제작한다.
- 회원 및 상품 정보를 관리할 수 있는 백오피스를 제작한다.
- MSA 환경으로 개발한다.

### 🗓️ 개발기간/인원

---

- 2021.11 ~ 2021.12 / 5주
- 3인 프로젝트

### ⚙️ 개발환경

---

- Vue.js
- Thymeleaf
- Springboot
- Mybatis
- Oracle

### ⚡ 설명

---

![헤더](https://user-images.githubusercontent.com/23237567/147749638-7d581a88-67ec-4726-be0c-efb89cb51f68.gif)

- 헤더 (header)
    - 사이드바 열기
        - 좌측 상단의 버튼을 클릭하면 사이드바가 튀어나온다.
    - 메인페이지 이동
        - 현재 페이지의 정보를 표시한다.
    - 페이지 표시
        - 중앙의 로고를 클릭하면 메인 페이지로 이동한다.
    - 이벤트/신상품/베스트상품 탭
        - 메인 페이지에서는 각각 이벤트/신상품/베스트상품 페이지로 이동할 수 있는 탭이 표시된다.

![푸터](https://user-images.githubusercontent.com/23237567/147818981-20157424-d396-4677-a2b6-bef4d756f4fa.gif)

- 푸터 (footer)
    - 페이지 이동
        - 대부분의 페이지에서 홈/위시리스트/주문내역/마이페이지 이동 버튼을 제공한다.
    - 상품 옵션 선택
        - 상품 상세정보 페이지에서 구매하려는 상품의 옵션을 선택할 수 있는 버튼을 제공한다.
    - 구매하기
        - 장바구니 페이지에서 상품 결제 페이지로 이동할 수 있는 구매하기 버튼을 제공한다.
    - 결제하기
        - 상품 결제 페이지에서 상품을 구매할 수 있는 결제하기 버튼을 제공한다.

![사이드바](https://user-images.githubusercontent.com/23237567/147819015-6ebc481b-05e1-42e7-bb10-e18ae0d0f75f.gif)

- 사이드바
    - v-tab을 활용한 디자인
        - 카테고리가 3중 구조이므로 List로만 구현하면 사용자 입장에서 복잡할 수 있다고 판단
        - 따라서 대분류를 탭으로 선택할 수 있도록 구현
    - v-if를 활용한 조건부 렌더링
        - 사이드 바의 하단에 현재 로그인 여부에 따라 로그인 로그아웃 버튼이 다르게 나타남

![로그인](https://user-images.githubusercontent.com/23237567/147819032-80fae879-4c5b-43c1-8d12-d17f4cb3d32f.gif)

- 로그인
    - 기능 설명
        - 회원과 관련된 모든 기능은 클릭 시 로그인 화면으로 이동
        - 회원 DB에 있는 정보로 로그인 시 회원 서비스 이용 가능함
    - 기술 설명
        - Vuex를 사용하여 로그인 정보를 상태에 저장함
        - 브라우저의 session storage에도 저장하여 상태가 초기화되었을 때 복원

![마이페이지](https://user-images.githubusercontent.com/23237567/147819041-931bf514-d5a6-4fbb-9185-686050a22709.gif)

- 마이페이지
    - 사용자 정보 조회
        - 로그인 된 사용자의 정보를 조회할 수 있다
    - 보유 포인트 및 쿠폰
        - 사용자가 보유 중인 포인트 및 쿠폰 현황을 확인할 수 있다.
    - 주문 배송 조회
        - 사용자가 주문한 상품들의 배송 현황을 확인할 수 있다.

![위시리스트](https://user-images.githubusercontent.com/23237567/147819839-774e9571-3323-41d4-967f-bf3740b0aa3e.gif)

- 위시리스트
    - 기능 설명
        - 상품의 ❤️ 아이콘 클릭 시 해당 상품이 위시리스트에 담긴다.
        - 위시리스트에 담긴 상품을 다시 클릭하면 위시리스트에서 삭제된다.
    - 기술 설명
        - Product Item 컴포넌트 사용
        - Vuex-persistedstate 라이브러리 사용

![상품목록](https://user-images.githubusercontent.com/23237567/147819447-f7fc46b7-af87-4130-aafb-1be8d7113c56.gif)

- 상품 목록
    - 상품 전시
        - 선택한 카테고리에 해당하는 상품들을 조회할 수 있다.
    - 위시리스트 추가
        - 하트 버튼을 클릭하면 해당 상품을 위시리스트에 추가 제거할 수 있다.
    - 상품 이미지 변경
        - 이미지 하단의 컬러칩 버튼을 클릭하면 상품의 이미지를 변경할 수 있다.
    - 페이징
        - 한 페이지에 12 개씩 상품 정보를 받아오도록 페이징 기능이 제공된다.

![상품상세](https://user-images.githubusercontent.com/23237567/147819553-d34fdbc6-1b8d-4b13-a1b8-9042f8d65dbf.gif)

- 상품 상세정보
    - 상품 정보
        - 상품의 상세정보를 조회할 수 있다.
    - 상품 이미지 변경
        - 이미지 하단의 컬러칩 버튼을 클릭하면 상품의 이미지를 변경할 수 있다.
    - 상품 옵션 선택
        - 푸터의 옵션 변경 버튼을 클릭하면 구매하려는 상품의 색상과 사이즈를 선택할 수 있다.
    - 장바구니 담기
        - 색상, 사이즈, 개수를 모두 입력하면 하단의 구매 버튼이 활성화 된다.

![이벤트](https://user-images.githubusercontent.com/23237567/147820007-5a128950-74f9-4782-9545-f25d34101834.gif)

- 이벤트 배너 및 상세정보
    - 기능 설명
        - 전시관리 된 순서대로 이벤트 이미지가 보인다.
        - 3초마다 슬라이드가 전환된다.
        - 이벤트 이미지 클릭 시 해당 이벤트 상세화면으로 전환된다.
        - 쿠폰이 발급된다.
        - 예외 발생 시 스낵바가 띄워진다.
    - 기술 설명
        - 쿠폰 발급 시 회원 DB 에서 Transaction 처리
            - COUPON TABLE 쿠폰 생성
            - EVENT TABLE 이벤트 쿠폰 수량 감소

![신상품](https://user-images.githubusercontent.com/23237567/147819739-63bae871-97a5-43b6-b98d-cdebe46b2a1c.gif)

- 신상품
    - 신상품 선정 기준
        - 가장 최근에 등록된 상품이 신상품으로 선정된다.
    - 상품 전시
        - 중분류별 5개의 최신 상품을 조회할 수 있다.

![베스트상품](https://user-images.githubusercontent.com/23237567/147819459-cdd3f4a4-c9a6-4b00-8b37-2884d1b39a27.gif)

- 베스트상품
    - 베스트상품 선정 기준
        - 판매량이 많고 리뷰 점수가 높은 상품이 베스트상품으로 선정된다.
    - 상품 전시
        - 중분류 별 1위부터 5위까지의 베스트 상품을 조회할 수 있다.

![장바구니](https://user-images.githubusercontent.com/23237567/147819102-ee690a36-3abe-42ff-bd47-63d1ff2949e1.gif)

- 장바구니
    - 기능 설명
        - 옵션 변경 버튼을 누르면 해당 상품의 컬러와 사이즈 베리에이션을 보여줌
        - 상품 색상, 사이즈, 수량 변경 가능
        - 전체 선택, 전체 해제 기능 구현
        - 체크박스를 선택할 때마다 해당 상품의 가격 * 수량의 합계가 즉시 나타나게 함
        - 체크박스 1개 이상 선택해야 주문하기 버튼 활성화
        - 선택 삭제, 단일 삭제, 선택 구매, 단일 구매 가능
    - 기술 설명
        - 2번의 REST API(회원 DB와 상품 DB) 통신
        - 체크박스 v-model로 양방향 바인딩한 후 watch로 체크박스 변경 감시
        - 체크박스에 담긴 상품의 인덱스를 바탕으로 상품의 가격을 계산하여 데이터 바인딩

![주문하기](https://user-images.githubusercontent.com/23237567/147819081-9c57c191-bb63-475a-940e-3362e5b40e74.gif)

- 결제 페이지
    - 기능 설명
        - 사용자, 상품, 보유 쿠폰, 포인트 등의 정보를 확인할 수 있다.
        - 모달 창을 통해 배송지와 보유 쿠폰 정보를 변경할 수 있다.
        - 포인트와 결제 정보를 선택할 수 있다.
        - 결제 버튼 클릭 시 주문 완료 로딩 컴포넌트가 3초 유지된다.
        - 결제 완료 페이지로 전환된다.
        - 결제 버튼 클릭 시 결제 정보가 저장된다.
    - 기술 설명
        - 분리된 DB에 결제 정보 저장
        - 각각의 DB에서 Transaction 사용
        - 배송지, 쿠폰, 주문 완료 시 로딩 컴포넌트 사용

![주문내역](https://user-images.githubusercontent.com/23237567/147819106-2057ee02-fb1a-4c73-b726-12c50f51f0fb.gif)

- 주문내역
    - 날짜 선택하고 검색 시 기간에 맞는 odate를 가진 주문내역만 목록에 검색되어 출력됨
    - 주문 번호 내림차순으로 정렬

![주문상세내역](https://user-images.githubusercontent.com/23237567/147819110-88ffd52b-4888-4472-b646-aa8c0813326c.gif)

- 주문 상세정보
    - OrderItem 컴포넌트로 분리
    - 재사용 가능

![상품 리뷰](https://user-images.githubusercontent.com/23237567/147819127-86fd9c04-3fff-4c9d-9fb6-b05b6c055325.gif)

- 상품 리뷰
    - 리뷰 작성 기준
        - 구매한 상품에 한해 리뷰 작성이 가능하며, 작성된 리뷰는 해당 상품의 상세정보 페이지에서 확인할 수 있다.
    - 리뷰 기능
        - 별점
            - 최대 5점의 점수를 부여할 수 있다.
        - 간단 리뷰
            - 만족/보통/불만족 중 하나를 선택할 수 있다.
        - 상세 리뷰
            - 줄글로 리뷰를 작성할 수 있다.
    - MD 답변
        - MD는 백오피스에서 해당 리뷰에 답변이 가능하다.
